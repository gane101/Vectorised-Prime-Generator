This is a prime number generator which uses Wheel Factorization to find the prime numbers. https://en.wikipedia.org/wiki/Wheel_factorization

Here, we make a vector and keep adding the primorial number chosen at first to get new batches at each itereation. And those batches are checked using all the prime numbers found so far.
The number have to be chosen carefully. Higher primorial doesnt always mean faster performance.

To use it, you enter the file name, chosen primorial number and the target(All the primes upto the target will be checked)
